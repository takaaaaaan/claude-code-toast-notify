# Publishing to the VS Code Marketplace

So the extension shows up in the VS Code Extensions search.

## 1. Create a publisher

1. Sign in at <https://marketplace.visualstudio.com/manage> with a Microsoft account.
2. "Create publisher" — pick an ID (e.g. `takaaaaaan`).
3. Make sure `package.json` `"publisher"` matches that ID exactly.

## 2. Get a Personal Access Token (PAT)

1. Create a free org at <https://dev.azure.com/>.
2. User Settings → Personal Access Tokens → New Token.
3. Scope: **Marketplace → Manage**. Copy the token.

## 3. Publish

```powershell
cd vscode-extension
npx --yes @vscode/vsce login <publisher>   # paste the PAT
npx --yes @vscode/vsce publish              # or: publish minor / patch to bump
```

Appears in VS Code search within a few minutes. Users install via the
Extensions view or `ext install <publisher>.terminal-focus`.

## Notes

- **Naming / trademark**: keep the publisher and extension id neutral (your own
  handle). "Claude Code" is used only descriptively (nominative) in the
  description/keywords — do not imply an official Anthropic product.
- **Icon**: `icon.png` (128×128) is generated by `make-icon.ps1`. Replace it with
  your own to rebrand.
- **Open VSX** (for VSCodium / Cursor users): `npx --yes ovsx publish -p <token>`.
